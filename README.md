# Домашнее задание: Работа с событиями

## Задание

Вам необходимо реализовать несколько функций для работы с событиями DOM и интерактивной каруселью.

### Задание 1: Управление событиями

Реализуйте три функции для работы с событиями элементов:

#### 1. `addEventsById(elementId, events)`

Функция добавляет обработчики событий к элементу по его ID.

**Параметры:**
- `elementId` (string) - ID элемента, к которому нужно добавить события
- `events` (array) - массив строк с названиями событий (например, `['click', 'mouseover']`)

**Требования:**
- Если элемента с указанным ID не существует, функция должна завершиться без ошибок
- Для каждого события создайте обработчик, который:
  - Увеличивает счетчик срабатываний события (хранится в `element.dataset[eventName]`)
  - Вызывает функцию `updateStatus(elementId)` для обновления статуса
- Обработчики должны сохраняться в глобальном объекте `handlers`:
  ```javascript
  handlers[elementId] = {
    click: [handler1, handler2, ...],
    mouseover: [handler1, ...]
  }
  ```
- При повторном вызове функции с теми же событиями, обработчики должны добавляться (не заменяться)

#### 2. `removeEventsById(elementId, events)`

Функция удаляет обработчики событий у элемента по его ID.

**Параметры:**
- `elementId` (string) - ID элемента, у которого нужно удалить события
- `events` (array) - массив строк с названиями событий для удаления

**Требования:**
- Для каждого указанного события:
  - Удалите все обработчики этого события с элемента
  - Очистите массив обработчиков (сделайте его пустым массивом)
- **Важно:** счетчики срабатываний должны сохраняться после удаления обработчиков
- После удаления вызовите `updateStatus(elementId)` для обновления статуса


### Задание 2: Интерактивная карусель

Реализуйте функцию `setupCarousel(carouselWrapperSelector, modalSelector)` для настройки карусели с модальным окном.

**Параметры:**
- `carouselWrapperSelector` (string) - CSS селектор обёртки карусели
- `modalSelector` (string) - CSS селектор модального окна

**Требования:**

1. **Делегирование клика на элементы карусели:**
   - За карусель отвечает селектор `.carousel`
   - При клике на элемент с классом `.item` должно открываться модальное окно
   - Проверьте, что элемент действительно находится внутри карусели

2. **Закрытие модалки при клике вне:**
   - Модалка должна закрываться если:
     - Модалка открыта (не содержит класс `hidden`)
     - Клик был вне модалки
     - Клик был вне карусели
   - Модалка НЕ должна закрываться при клике внутри модалки или на элементы карусели

3. **Прокрутка карусели стрелками:**
   - Добавьте обработчики на кнопки `.arrow.left` и `.arrow.right`
   - При клике на левую стрелку: прокрутите карусель влево на 200px
   - При клике на правую стрелку: прокрутите карусель вправо на 200px

**Важно:** Если обёртка карусели не найдена, функция должна завершиться без ошибок.
